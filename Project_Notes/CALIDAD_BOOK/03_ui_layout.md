## Telegram-бот (только для Руководителя)

**Назначение:** быстрые действия с телефона без открытия веб-приложения.

**Доступно через Telegram:**
```
🔔 Уведомление: «Аудит завершён: найдено 3 дубля МП»
   [Показать детали] [Открыть в браузере]

🤖 Предложение агента: «МП-83 и МП-91 похожи. Объединить?»
   [Да] [Нет] [Показать подробнее]

🎙️ Голосовая заметка: наговорить контекст → агент транскрибирует

💬 Текстовый вопрос агенту: быстрые запросы к базе знаний
```

**НЕ делается через Telegram** (только в веб):
- Редактирование текстов МП
- Просмотр сложных таблиц и отчётов
- Управление шаблонами

**Голос → текст:** Telegram Premium автоматически транскрибирует голосовые в текст прямо в чате. Руководитель копирует готовый текст — никакого STT-сервиса не нужно.  
**Защита:** пароль при первом запуске бота.  
**Технический стек:** telegraf.js + Cloud Run webhook.  
**Приоритет:** после того как веб-функционал агента устоится. Telegram снижает фактор «бутылочного горлышка» — руководитель одобряет с телефона без открытия браузера.
