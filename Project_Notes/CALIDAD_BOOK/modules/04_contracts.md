## Процесс

```
Руководитель (ContractBuilder):
  1. Создаёт HTML-шаблон с переменными {{buyer}}, {{amount}}
  2. Управляет библиотекой версий (поставка, аренда, услуги)

Менеджер (ContractSidebar):
  1. Вводит реквизиты покупателя
  2. Добавляет спецификацию
  3. Наносит печать/подпись drag-and-drop
  4. Генерирует PDF
```
