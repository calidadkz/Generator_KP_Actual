# 03. КП-генератор

**Статус:** ✅ Реализован  
**Роли:** Руководитель (создаёт шаблоны), Менеджер (заполняет и генерирует)  
**Связи:** [[INDEX]] [[modules/16_closingscript]] [[02_roles_access]]

---

## Процесс

```
Руководитель (TemplateMapper):
  1. Загружает дизайнерский PDF-бланк
  2. Рисует прямоугольники drag-to-draw над полями
  3. Задаёт имя переменной (price, client_name...)
  4. Сохраняет шаблон → Zustand + localStorage

Менеджер (GeneratorPage):
  1. Выбирает шаблон из галереи
  2. Вводит значения для каждой переменной
  3. Настраивает форматирование (шрифт, цвет, выравнивание)
  4. Видит live-превью
  5. Скачивает PDF
```

---

## Ключевые технические детали

- PDF-координаты: pdf-lib использует origin (0,0) внизу слева, A4 = 595×842pt
- Canvas: origin (0,0) вверху слева → конвертация через `canvasRectToPdfBox()`
- Шрифты: только WOFF с кириллицей (`/public/fonts/`) — Arial/Helvetica не работают
- PDF-бланки хранятся в IndexedDB (не localStorage)
