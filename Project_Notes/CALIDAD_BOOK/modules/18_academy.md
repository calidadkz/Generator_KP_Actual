# 18. Академия CALIDAD — Видеотека с RAG-поиском

**Статус:** 💡 Будущее (концептуально описан, не начат)
**Роли:** Менеджер (обучение), Руководитель (управление контентом)
**Связи:** [[modules/05_cockpit]] [[modules/07_agent]] [[modules/10_trainer]]

---

## Концепция

Библиотека видеоматериалов по продукту и продажам с умным поиском. Менеджер не листает каталог — он спрашивает «как объяснить клиенту охлаждение» и сразу попадает на нужный момент в нужном видео.

**Аналогия:** YouTube + поиск внутри расшифровок → Академия по конкретному продукту.

---

## Два типа контента

### 1. Обучающие материалы (внутренние)

Видео и записи которые создаёт сам Евгений:
- Разборы звонков (реальные кейсы с комментариями)
- Тренинги менеджеров (уже есть записи — Type B диалоги)
- Технические обзоры станков
- Методики работы с возражениями

### 2. Внешние ресурсы (ссылки)

Ссылки на YouTube-видео возможностей станков — уже используются для «прогрева» клиентов:
```
Менеджер → «просто интересующийся» клиент →
  «Посмотрите вот это видео возможностей лазера [ссылка]»
  «И вот это по фрезеру [ссылка]»
  → через 10 минут: «Какое видео больше зашло?» → квалификация по реакции
```

---

## RAG-поиск по расшифровкам

Каждое видео → расшифровка → индексируется → поиск по смыслу, не по тегам.

```
Менеджер в Кокпите: 🔍 «резка кожи особенности»
→ Академия находит:
  📹 «Лазерные станки часть 2» → 14:32 — говорится про кожу
  📹 «Лазерные станки CO2» → 8:05 — параметры для кожи
→ Кликает → видео открывается с нужного момента
```

Та же технология что в [[07_agent]] — эмбеддинги + Firestore vector search.

---

## Связь с Кокпитом

В режиме «Справочник» (переписка):
```
🔍 «видео резка кожи» → ссылка из Академии + таймстамп
```

В режиме Нестандартная задача:
```
«Клиент спрашивает про резку поролона» →
Академия: «Не найдено видео по поролону» →
Создаётся задание: «Записать видео-разбор кейса с поролоном»
```

## Firestore

Коллекция `academy_videos`:
```typescript
AcademyVideo {
  id: string;
  title: string;
  type: 'internal' | 'youtube_link';
  url: string;
  transcript?: string;      // расшифровка для RAG
  tags: string[];
  machineTypeIds?: string[]; // к каким станкам относится
  durationSec?: number;
  addedAt: Timestamp;
}
```
