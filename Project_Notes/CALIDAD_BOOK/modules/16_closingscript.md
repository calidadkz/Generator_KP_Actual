## Скрипт Дожима (из документа)

### Когда используется
Идёт после: Квалификация → подготовка КП → отправка → **Дожим**.

### Ключевая особенность
Скрипт Дожима помнит контекст Квалификации.
При открытии — сводка ИИ сверху:
`"Клиент сомневается из-за габаритов. Заложен чиллер 5200. Бюджет впритык."`

### Правая панель (МП для Дожима)
Другой набор МП чем в Квалификации:
- Методики компромиссов: «Убрать 4-ю ось, оставить высокий портал на будущее»
- Аргументы против конкурентов: «Почему в Панде дешевле (алюминиевый профиль)»
- Экономика владения, гарантии, отзывы

### Режим «Закрытие на инженера»
Кнопка `[Нестандартная задача]` → скрипт выдаёт формулировку для передачи лида Евгению.

---

## Как собирается скрипт Дожима

Так же как скрипт Квалификации — из диалогов с помощью агента:
1. Агент собирает эффективные диалоги по Дожиму
2. Извлекает этапы, МП, варианты ответов клиентов
3. Руководитель настраивает в Admin Panel

---

## Библиотека для Дожима

Библиотека знаний **не заменяется** для Дожима — она содержит всё. Нужна настройка анализа отдельно для этапа дожима.
