## Процесс (4-шаговый мастер)

```
1. Тема — выбор темы (из BatchInsights или вручную)
2. Черновик — AI генерирует текст статьи
3. Рестайл — переписывает в стиле StyleDNA автора
4. Публикация — сохранить в базе / опубликовать в WordPress
```

---

## StyleDNA

Извлекается из диалогов Евгения: частые фразы, средняя длина предложения, тон, структура мысли.
Используется для рестайлинга статей — «голос автора».

Хранится в Firestore: коллекция `style_dna`.

---

## FewShot примеры

Образцовые примеры стиля для рестайлинга. Коллекция `few_shot_examples`.

## WordPress

Публикация через REST API + Application Passwords.
Индикатор подключения в шапке Dashboard (проверка каждые 30 сек).
