# API: стоимость и оптимизация

**Связи:** [[modules/07_agent]] [[01_architecture]]

---

## Модели по задачам

| Задача | Модель | Почему |
|--------|--------|--------|
| RAG поиск, простые задачи | Gemini Flash | Дёшево, быстро |
| Агент-администратор (чат) | Claude Sonnet | Качество рассуждений, tool use |
| Тренажёр: ведение сессии | Claude Haiku + кэш | Достаточно для роли, кэш даёт экономию |
| Тренажёр: оценка сессии | Claude Sonnet | Сложный анализ по чек-листу |
| Оценка реальных звонков | Claude Sonnet | Сложный анализ |
| Генерация черновиков МП | Claude Haiku / Gemini Flash | Достаточно |

---

## Главная экономия: Prompt Caching

Системный промт агента (описание CALIDAD, все коллекции, правила) одинаков в каждом запросе.  
Claude API: повторяющееся начало промта кэшируется → платишь 10% от обычной цены.

**Экономия: 60-80% стоимости агента.**

---

## Ориентир стоимости

Для команды 3-5 менеджеров (с кэшированием):

| Функция | Стоимость/месяц |
|---------|----------------|
| Тренажёр (6 сессий/неделю) | ~$2 |
| Оценка реальных звонков (10/неделю) | ~$1 |
| Агент-администратор | ~$4 |
| Рутины и фоновые задачи | ~$1 |
| **Итого** | **~$8-15** |

При 10 менеджерах: ~$30-50/месяц.  
Без кэширования — умножать на 3-4.

## Важное разъяснение

Подписка Claude Pro/Max ($20-100/месяц) — для chat.claude.ai, **не для API**.  
API всегда оплачивается по токенам отдельно.  
Для приложений используется Anthropic API (отдельный аккаунт).
